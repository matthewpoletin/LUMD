# LUMD

## Brief
LU matrix decomposition in C with single thread, multiple threads, MPI and OpenMP

## Setup
```bash
    $ git clone https://github.com/matthewpoletin/LUMD.git
    $ cd LUMD
    $ git submodules init
```

## Execution
### Single thread
```bash
    $ cmake .
    $ make
    $ ./LUMD
```
